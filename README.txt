== Description ==
The UserLast plugin is used to display the date and time of the last login of users of your wordpress website.
 
After starting UserLast, information about the login time will be displayed in two places:
 
*   In the new column \"last login\" on the list of all users in the administration panel. It is possible to sort the list of users by logging in.
*   In the edit view of a specific user in the admin panel, in the \"Last login\" section.
